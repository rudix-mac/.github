# Rudix Project – End of Life (April 1, 2026)

After 21 years, the Rudix project is being retired. Mission accomplished.

Rudix was born in 2005 as a pioneering alternative for distributing precompiled binary packages on macOS, long before Homebrew became popular. It served users and developers worldwide with universal binaries for PowerPC and x86 architectures.

The decision to end development comes from two main factors: Apple's transition to ARM64 architecture with Apple Silicon, and the high cost of acquiring new development hardware in Brazil. Maintaining the quality standards the project demands is no longer sustainable.

All repositories under the rudix-mac organization on GitHub will be archived as of April 1, 2026. No further development or updates will be made.

The project website at http://rudix.org will eventually transition to a personal site and will no longer serve as the project's homepage. The historical archives will remain on GitHub.

I ask the community to respect the name "Rudix" as my personal property. It is a name I use as a personal moniker, and I hope it will be treated with the respect it deserves.

Thank you to everyone who contributed, supported, and used Rudix throughout these two decades. It was hard work, filled with pride and satisfaction.
